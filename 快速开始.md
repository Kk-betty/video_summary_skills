## 🎯 三种使用方案

### 方案一：局域网访问（最简单，推荐）

如果您和朋友在**同一个WiFi网络**下，直接使用：

```
http://192.168.1.5:5174
```

**无需任何配置，直接分享此链接即可！**

---

### 方案二：使用我们的内网穿透工具

如果需要在**任何网络**访问，请按以下步骤：

1. **在项目目录打开一个新终端**
2. **运行以下命令：**

```bash
node tunnel.js
```

3. **等待几秒钟**，您会看到类似这样的输出：
   ```
   🌐 您的外网访问链接：
      https://happy-cat-123.loca.lt
   ```

4. **将这个链接分享给朋友**

---

### 方案三：手动安装和运行

如果方案二遇到问题，请尝试：

1. **在项目目录打开终端**
2. **运行：**

```bash
node -e "const { exec } = require('child_process'); exec('npm install localtunnel', (err) => { if (!err) { const lt = require('./node_modules/localtunnel'); lt({port:5174}).then(t => console.log('✅ 外网地址:', t.url)); } });"
```

---

## 📁 可用的启动脚本

| 脚本 | 用途 |
|------|------|
| `tunnel.js` | 启动内网穿透（推荐） |
| `start-all.js` | 一键启动前端+后端 |
| `simple-tunnel.js` | 简单版隧道脚本 |

---

## 💡 推荐使用顺序

1. **首先尝试** → 方案一（局域网，最简单）
2. **如果需要外网** → 方案二（`node tunnel.js`）
3. **如果方案二不行** → 方案三（手动方式）

---

## ⚠️ 注意事项

- 🔒 保持运行内网穿透的终端窗口不要关闭
- 🔄 每次重新启动内网穿透，链接可能会变化
- 🛡️ 分享链接时注意不要泄露敏感信息
